# Kantin Kejujuran BPS Kota Gunungsitoli

Website sistem informasi Kantin Kejujuran BPS Kota Gunungsitoli berbasis web menggunakan CodeIgniter 3 dan Bootstrap 5.

## ✨ Fitur Utama

- Autentikasi Login & Register
- Dashboard untuk Admin dan User
- CRUD Produk (Admin)
- CRUD User (Admin)
- Laporan Transaksi
- Desain modern dengan tema Glassmorphism
- Sidebar navigasi (bukan navbar atas)

## 🛠️ Teknologi yang Digunakan

- PHP 7.x
- CodeIgniter 3
- Bootstrap 5
- MySQL
- JavaScript
- HTML/CSS (Glassmorphism UI)

## 🗂️  Database

Database: `kantin_kejujuran`

## Selengkapnya...

👥 Role Pengguna
Admin: Kelola produk, user, dan laporan transaksi.
User: Melihat dan membeli produk.

🧊 Desain UI
Tema desain Glassmorphism dengan elemen transparan, blur, dan bayangan halus.
Sidebar navigasi menggantikan navbar klasik.

🔐 Keamanan
Password terenkripsi menggunakan password_hash()
Validasi input pada form login dan register
